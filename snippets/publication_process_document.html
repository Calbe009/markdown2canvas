Quick search

  1. The style is applied.

    1. The markdown header and footer from the used style are assembled around your source.md.

    2. The html header and footer from the used style are assembled around that.

  2. Replacements are done.

  3. Emoji shortcodes are emojified

  4. The markdown is translated to HTML.

  5. Links to existing content are implemented, further modifying the HTML.

  6. Local files and images which are linked-to in the source are uploaded (if necessary), and the HTML is adjusted to use the Canvas links to that content.

  7. Content properties are set from meta.json.

  8. The content is placed in modules as described in meta.json.
